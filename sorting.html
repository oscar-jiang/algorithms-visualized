👋About

This sorting algorithm visualizer was created to help users understand how common sorting techniques operate step by step. Built using JavaScript and the D3.js library, this interactive tool visually demonstrates popular sorting algorithms including Bubble Sort, Merge Sort, Quick Sort, and Selection Sort.

Users can customize the number of elements to sort, adjust playback speed, and switch between different algorithms to compare performance and behavior.

Although there are a lack of settings, more features are coming!
